# type-tuple-literal

## Syntax

_tuple_literal_ :=\
      `(` `,` `)`\
    __|__ `(` [_type-expression_] `,` `)`\
    __|__ `(` [_type-expression_] __(__ `,` [_type-expression_] __)+__ `,`__?__ `)`

[_type-expression_]: type_expression.md

## Semantics

A _type-tuple-literal_ creates a tuple of types.
